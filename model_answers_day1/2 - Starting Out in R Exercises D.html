Exercises D.
letters is an inbuilt vector that contains the letters of the alphabet.

print(letters)
[1] "a" "b" "c" "d" ...

Write R code to make the following selections:

  1. Letter 18 in the alphabet.
letters[18]
[1] "r"
  1. The first three letters in the alphabet.
letters[1:3]
[1] "a" "b" "c"
  1. Just letters in positions 1, 5, 9, 15, 21.
letters[c(1,5,9,15,21)]
[1] "a" "e" "i" "o" "u"
  1. All letters except those in positions 1, 5, 9, 15, 21.
letters[-c(1,5,9,15,21)]
 [1] "b" "c" "d" "f" "g" "h" "j"
 [8] "k" "l" "m" "n" "p" "q" "r"
[15] "s" "t" "v" "w" "x" "y" "z"
  1. The last 3 letters in the alphabet.
letters[24:26]
[1] "x" "y" "z"
  1. Can you:
    a. Repeat part 5. so that it would also work to select the last 3 items of a differently length list?
n <- length(letters)
letters[(n-2):n]
[1] "x" "y" "z"

or

letters[(length(letters)-2):length(letters)]
[1] "x" "y" "z"

"h", "e", "l", "l", "o", "w", "o", "r", "l", "d"
